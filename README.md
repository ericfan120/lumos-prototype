# Lumos Media Planning Platform

Interactive prototype for the Lumos Media Planning Platform - an AI-powered media scenario planning and optimization tool.

## Live Demo

Visit the live demo at: `https://YOUR_USERNAME.github.io/lumos-prototype/`

## Features

- **Brief & Strategy** - Campaign overview, Communications Architecture, Goal Setting & Measurement Framework
- **Performance** - KPIs, Channel Mix, Reach Curves, Challenge Diagnosis
- **Channel Mix** - Detailed channel allocation and metrics
- **Flighting & Calendar** - Multi-media flighting optimization with cultural calendar
- **Mapping** - Geographic investment distribution with channel breakdown by region
- **Partners** - Publisher recommendations and activation status

## Technology

- Vanilla HTML/CSS/JavaScript
- Tailwind CSS (CDN)
- Custom SVG visualizations
